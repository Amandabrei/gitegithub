Colocando,
projetos no git

Primeiro: faça uma pasta no computador com o nome do projeto, depois araste ela para o VSC.

Primeiro Passo

dentro do VSC crie uma index.html ou coloque o nome que quiser, digite ! e de um enter para completar o código, coloque pt-br, e faça as alterações para começar o projeto.
. para colocar seu nome digite: meta name="author" content="Amanda Bastos">
. para trabalhar com Css digite: link rel="stylesheet" href="caminho do css">
Para iniciar o git, após começar o projeto clique em Viwe - Terminal .digite: git init (para inicializar o repositório no git)
.para ver o status digite: git status (veremos que no inicio ele não tem commits yet(agora))
.para começar a trabalhar no Git digite: git add index.html
.após isso veremos q ele esta preparado para ser committed, ele foi adicionado
.para criar a primeira versao digite: git commit -m "inicializamos o projeto"
.a primeira vez tem que cadastrar o email e usuario, basta copiar a linha do email e colar colocando seu email - faça o mesmo no user
.dai já pode novamente criar a primeira versao digitando: git commit -m "inicializamos o projeto"
.para logar o projeto digite: git log (vai mostrar tudo sobre o projeto)

Segundo Passo

Quando você cria novas pastas, como css, voce tem que dar comando curinga para add tudo que foi alterado, para isso digite: git add .
O bom é ir criando legendas a cada motificação, git commit -m "adicionei css"
Não esquecendo do git log - que irá mostrar todas histórico do projeto.
- para saber qual commit estamos no momento digite: git log
- para voltar para uma versão anterior digite: git checkout e cole o commit anterior assim: git checkout 1aa04b74f1e480d4eb035a5d057e91cea5357762
- no caso acima a versao não foi deletada, você apenas está no commit anterior.
- para ver todos commits digite novamente: git log --all (dai irá mostrar todos os commits, você apenasmudou o commit)
- para voltar para a ultima versão digite: git checkout master
